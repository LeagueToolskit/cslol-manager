# 🎨 CSLoL Manager - League of Legends Skin Manager

Modern, fast, and user-friendly skin manager for League of Legends. Built with Electron, React, and TypeScript to provide a seamless skin browsing and installation experience.

## ✨ Features

* 🎨 **Browse & Download Skins** – Access a large collection of community-made skins.
* 🔍 **Advanced Search & Filtering** – Quickly find skins using categories and search tools.
* ⚡ **One-Click Installation** – Install skins automatically with minimal setup.
* 🌍 **Multi-Language Support** – Available in multiple languages with more coming soon.
* ⭐ **Favorites System** – Save and organize your favorite skins.
* 🔄 **Auto Updates** – Receive the latest features and improvements automatically.
* 🎮 **Automatic Game Detection** – Detects your League of Legends installation instantly.
* 🖥️ **Modern Interface** – Clean, responsive, and easy-to-use UI.

## 🚀 Installation

1. Download the latest release from the Releases page.
2. Run `CSLoLManager-Setup-x.x.x.exe`.
3. Follow the installation wizard.
4. Launch CSLoL Manager and start exploring skins.

## 🛠️ Development

### Requirements

* Node.js 20+
* pnpm (recommended) or npm
* Windows

### Setup

```bash
git clone https://github.com/yourusername/cslol-manager.git
cd cslol-manager

pnpm install
pnpm dev
```

### Build

```bash
pnpm build:win
```

## ⚙️ Configuration

Application data is stored in:

```text
%APPDATA%/CSLoLManager
```

Configurable options include:

* Language
* Theme
* Download Location
* Update Preferences
